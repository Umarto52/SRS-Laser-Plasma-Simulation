# SRS Laser Plasma Simulation

This repository contains Python simulations developed for the study of Stimulated Raman Scattering (SRS) in laser–plasma interactions.

## Research Topic

Nonlinear Evolution of Stimulated Raman Scattering in Laser–Plasma Interactions: Electron Trapping, Kinetic Inflation and Broadband Laser Effects

## Python Scripts

- plasma_frequency.py
- srs_growth_rate.py
- broadband_suppression.py
